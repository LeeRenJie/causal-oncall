How this event lands in Grail

1. Orchestrator finalises the brief.
2. DynatraceClient.send_investigation_event() ingests this envelope
   via the Events API.
3. Dynatrace correlates on `dt.entity.problem` and pins the event to
   the problem's timeline.
4. The on-call sees the investigation alongside their own notes.
